BetAI GitHub Pages

Jak wrzucić:
1. Rozpakuj ZIP.
2. Wrzuć całą zawartość do nowego repozytorium GitHub.
3. Wejdź w Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Zapisz.

Pliki:
- index.html
- 404.html
- .nojekyll
- assets/

Dlaczego jest 404.html:
GitHub Pages czasem pokazuje 404 przy wejściu bezpośrednio w niektóre ścieżki.
Ta kopia pomaga jako fallback.

Jeśli logowanie działa, ale liczniki pokazują 0:
uruchom plik SETUP_SUPABASE.sql w Supabase SQL Editor.
