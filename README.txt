BetAI PRO Live App

Pliki:
- index.html
- _redirects
- SETUP_SUPABASE.sql

Jak wrzucić:
1. Rozpakuj ZIP.
2. Wrzuć CAŁĄ zawartość na hosting statyczny.
3. Nie usuwaj pliku _redirects.

Jeśli logowanie działa, ale statystyki pokazują 0:
- otwórz Supabase SQL Editor
- wklej zawartość SETUP_SUPABASE.sql
- uruchom

W tej paczce:
- logowanie
- rejestracja
- wylogowanie
- sesja użytkownika
- licznik zarejestrowanych
- licznik aktywnych użytkowników
- dashboard po zalogowaniu
